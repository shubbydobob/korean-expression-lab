# Korean Learning Platform

Public-first Korean/English expression learning platform with an internal admin workspace and an AI prompt harness.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Vercel-ready deployment shape
- Prompt harness for structured AI tasks and evaluation

## Local Run

1. Copy `.env.example` to `.env.local`
2. Fill `ADMIN_EMAIL`, `ADMIN_PASSWORD`, and `SESSION_SECRET`
3. Run `npm run dev`
